## üìå Project Overview
The **Smart Health & Activity Tracker** is a robust Android application designed to help users track their daily health metrics. This project fulfills all the requirements of the MAD Open Ended Lab, including local storage, cloud authentication, and hardware sensor integration.

## ‚ú® Key Features & Functionalities

### 1. User Authentication (Bonus +3)
* Secure Login and Sign-Up system powered by **Firebase Authentication**.
* Ensures user data privacy and session management.

### 2. Real-time Step Counter (Advanced Feature)
* Utilizes the device's **Accelerometer Sensor** to track steps.
* Real-time UI updates with Start, Stop, and Reset functionalities.

### 3. Health Logging (SQLite Persistence)
* Local data storage for Weight, Water Intake, and Calories.
* **CRUD Operations:** Users can save their daily logs, which are stored permanently in an SQLite database.

### 4. History View (RecyclerView)
* Dynamic list display using **RecyclerView** and a Custom Adapter.
* Fetches and displays all historical logs from the local database.

### 5. Daily Health Quotes (REST API - Bonus +3)
* Integrates the **Quotable API** using the **Volley Library**.
* Fetches motivational health quotes dynamically from the internet.

### 6. Modern UI/UX (Bonus +4)
* **Dark Mode Support:** Automatically adjusts to system theme settings.
* **Material You UI:** Implements Material Design 3 guidelines for a modern look.
